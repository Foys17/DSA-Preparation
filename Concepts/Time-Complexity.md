# Time and Space Complexity & Space Complexity

## ✅ What is Time Complexity?

Time Complexity measures how the runtime of an algorithm grows as input size increases.

We usually express it using Big-O notation.

Examples:

O(1) → Constant time (fastest)
O(n) → Linear time
O(n²) → Quadratic time (slow)
O(2^n) → Exponential (very slow)

---

## ✅ What is Space Complexity?

Space Complexity measures how much extra memory an algorithm uses.

Example:

- Using a few variables → O(1)
- Creating new arrays → O(n)

---

## ✅ Why It Matters

Understanding complexity helps avoid:

- Slow solutions
- Time Limit Exceeded (TLE)
- Memory Limit Exceeded (MLE)
